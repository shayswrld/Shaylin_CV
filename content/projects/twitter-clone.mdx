A full-stack social media application built during my internship at Mohara Ventures, replicating core Twitter functionality using modern web technologies.

## Tech Stack:
- **T3 Stack**: Next.js, TypeScript, tRPC
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: NextAuth.js
- **Styling**: Tailwind CSS
- **Testing**: Jest for unit testing

## Features:
- User authentication and authorization
- Tweet creation and posting
- Real-time feed updates
- User profiles and following system
- Responsive design for mobile and desktop

## Learning Outcomes:
- Mastered full-stack development principles
- Gained experience with modern React patterns
- Learned database design and ORM usage
- Practiced test-driven development with Jest
